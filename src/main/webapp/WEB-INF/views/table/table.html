Possible options for entering data into a cell:

1) Just a number, for example: 12345678987654321

2) Mathematical expression in the form: =[math.expression],

for example: =1+2*(3+4/2-(1+-2))*2+1+(20.3+13)*(1+21)

3) SUM formula in the form: SUM([Range start]:[Range end])

for example: SUM(B2:B3)

Features:

- The beginning and end of the range can be swapped.

- Range must be in one column or one row (Do not go diagonally)

Links must link to filled cells
